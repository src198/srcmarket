# SRC Market
Маркетплейс юзернеймов и бейджиков в KConnect.

## Как установить?
1. Сначала скачайте репозиторий:
```bash
git clone https://github.com/src198/srcmarket.git
```
2. Установите зависимости:
```bash
npm install
```

3. Создайте билд:
```bash
npm run build
```

4. Запустите билд:
```bash
npm run start
```
